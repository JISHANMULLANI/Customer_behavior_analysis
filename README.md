# Customer_behavior_analysis
Data analytics project using python , sql , power Bi. 

👨🏻‍💻Customer Behavior Data Analyst Portfolio Project
This project represents a complete, industry standard, end-to-end data analytics workflow, designed to mirror the real responsibilities of professional analysts in modern business environments. The project encompasses all critical stages of data analysis, from data preparation and modeling to insight generation, visualization, and reporting.

This project is perfect for:
📊 Data Analyst aspirants who want to build a strong Portfolio Project for interviews and LinkedIn
📚 Anyone learning Python, SQL, and Power BI
💼 Professionals preparing for interviews in Data Analytics, Data Science or Product Analytics roles

## 🧰 Tech Stack

* Python  
* SQL 
* Power BI
* Jupyter Notebook

---

## 📁 Project Structure

```
Customer_behavior_analysis/
│
├── Customer_Shopping_Behavior_Analysis.ipynb   # Python analysis notebook
├── Database_information.sql                    # SQL queries for business insights
├── customer_behavior_dashboard.pbix            # Power BI dashboard
├── dataset/ustomer_behavior_dashboard.csv      # Dataset files
└── README.md
```

## ⚙️ How to Use This Project

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/JISHANMULLANI/Customer_behavior_analysis.git
cd Customer_behavior_analysis
```

---

### 2️⃣ Open the Jupyter Notebook

Open:

```
Customer_Shopping_Behavior_Analysis.ipynb
```

This notebook includes:

* 📥 Data Import
* 🔍 Data Exploration
* 🧹 Data Cleaning
* 🔗 Database Connection

---

### 3️⃣ Load Data into SQL Database

* Create a database in:

* MySQL 
* Run Python code from the notebook to:

  * Establish connection
  * Load cleaned data into SQL tables

---

### 4️⃣ Run SQL Queries

Open:

```
Database_information.sql
```

Use this file to:

* Answer business questions
* Analyze customer trends
* Extract insights from structured data

---

### 5️⃣ Power BI Dashboard

Open:

```
customer_behavior_dashboard.pbix
```

Steps:

* Connect Power BI to your SQL database
* Load required tables
* Build interactive visualizations
